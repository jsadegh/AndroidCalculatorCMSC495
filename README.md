# Compose Calculator (CMSC 495)

A modern calculator app for Android built with Kotlin and Jetpack Compose.  
Developed as part of the UMGC CMSC 495 Computer Science Capstone, with an emphasis on clean UI, reliable core functionality, and collaborative development using GitHub.

> Focus: readable code, predictable behavior, and a UI theme built around gold / plum / ink tones.

## Screenshot
![Calculator Screenshot](calculator_screenshot.png)

## Features
- Basic arithmetic: addition, subtraction, multiplication, division
- Decimal input with clean output formatting
- Division-by-zero handling (“Error” instead of crashing)
- Toggleable calculation history panel
- Compose-based UI using Material 3
- Consistent team-friendly commenting for collaboration

## Core Controls

| Button | What it does |
|-------:|--------------|
| AC     | Clears all input and resets the calculator |
| ⌫      | Deletes the last entered character |
| ⏱      | Toggles the calculation history panel |
| =      | Runs the current operation and shows the result |

## Built With
- Kotlin
- Jetpack Compose
- Material 3
- Android Studio + Gradle
